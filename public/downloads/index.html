How to install

  1. Tap Download APK above on your Android device.
  2. If prompted, allow downloads from the browser.
  3. Open the downloaded file. Android will ask to Install unknown apps — tap Settings, enable it, then go back and tap Install.
  4. Pair your HC-05 / HC-06 Bluetooth module in Android Settings before opening the app.

Requirements

  • 📱
    Android 7.0+
    API level 24 or higher
  • 📡
    Classic Bluetooth
    HC-05 or HC-06 beam sensor module
  • 🌐
    Wi-Fi (optional)
    App works offline and syncs when connected to realtimemaster.com
